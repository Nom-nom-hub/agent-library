# Support

## Getting Help

We want to make sure you get the help you need. Here are the ways to get support:

### Documentation

- **[README.md](./README.md)** - Quick start and overview
- **[PERSONAS.md](./PERSONAS.md)** - Understanding personas
- **[AGENTS.md](./AGENTS.md)** - Agent integration guide
- **[docs/](./docs/)** - Full documentation

### Questions & Discussions

If you have questions about how to use Agent Library:

1. Check the documentation and README first
2. Search existing [GitHub Issues](https://github.com/Nom-nom-hub/agent-library/issues)
3. Check [GitHub Discussions](https://github.com/Nom-nom-hub/agent-library/discussions)
4. Open a new discussion if you can't find an answer

### Bug Reports

If you found a bug:

1. Check existing [Issues](https://github.com/Nom-nom-hub/agent-library/issues)
2. Open a new issue with:
   - Clear description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - Your environment (OS, Python version, agent)

### Feature Requests

For feature suggestions:

1. Check existing [Issues](https://github.com/Nom-nom-hub/agent-library/issues)
2. Open an issue with:
   - Use case and motivation
   - Proposed solution (if you have one)
   - Alternative approaches
